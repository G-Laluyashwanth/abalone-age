# Abalone Age Prediction — Kaggle

Regression project predicting abalone age from physical measurements.

**Competition:** [Regression with an Abalone Dataset (Playground S4E4)](https://www.kaggle.com/competitions/playground-series-s4e4)

## Approach

Reused scikit-learn pipeline template from [ames-housing](https://github.com/G-Laluyashwanth/ames-housing) project — ColumnTransformer + SimpleImputer + OneHotEncoder + RandomForestRegressor.
